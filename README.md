AntibioticEnv System is a macOS-based scientific software designed for analyzing the occurrence, distribution, and ecological risks of antibiotics in aquatic environments.

It provides integrated data processing, risk assessment (RQ, PI, RRQ), and visualization tools for environmental researchers.
