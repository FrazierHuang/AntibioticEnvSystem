# AntibioticEnv System v1.1 (Windows, Bilingual, Welcome Wizard)

- First-launch welcome wizard with bilingual instructions
- Chinese/English toggle in the menu
- Outputs figures and Excel to `outputs/`

Build EXE on Windows:
1) Install Python 3.10+
2) pip install pyqt6 pandas matplotlib openpyxl pyinstaller
3) Run: build_windows.bat
